tmux-resurrectが入っている環境でtmuxがクラッシュした時、レジューム用のファイル（`~/.tmux/resurrect/last`）が正常に保存されない事があるようで、復元しようとした所、タイトルのようなエラーが出ました。

このような場合、`last`に対し既存のセッションファイルを手動でリンクし直してやる事で、解決できました。

```bash
$ cd ~/.tmux/resurrect
$ ln -sf tmux_resurrect_YYYY-MM-DDTHH:MM:SS.txt last
```
