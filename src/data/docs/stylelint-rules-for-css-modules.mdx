stylelint を導入したプロジェクトで CSS Modules を使う際、
rulesにmode切り替え用のセレクタ（[css-modules/css-modules#exceptions](https://github.com/css-modules/css-modules#exceptions)）が抜けているとエラーになるため、下記のように設定する。[^1]

## modeセレクタ用の登録

```js
 "rules": {
    ...
    "selector-pseudo-class-no-unknown": [
      true,
      {
        "ignorePseudoClasses": ["global", "local"]
      }
    ]
  }
```

[^1]: 手動の設定でなくとも、stylelint ruleset系のpluginにより解決できる可能性があります。
