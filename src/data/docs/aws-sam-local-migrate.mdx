aws-sam-localをaws-sam-cliにアップデート後、
template.ymlのSyntaxが変わっており、書き方次第では実行時エラーが発生します。

## FunctionNameにまつわるエラーが出る場合

ResourcesのFunctionNameに「-」等が混ざってはいけません

## Plugin 'ImplicitApiPlugin' raised an exception: 'Path'

Events内Propertiesの記法の問題です。<br />
下記の通り修正します。

- path -> Path
- method -> Method

Pythonのバージョン関係のエラーが出ている場合は、
2.7.x系 or 3.6にしてください。（公式リポジトリのサポートより）
